All Formulas

Calculated Columns

1. Date Diff

Calculates the number of days between chassis arrival and current date

Original Formula (SQL)
datediff(day, [ACTARRIVE], getdate())
DAX Formula
DATEDIFF([ACTARRIVE], TODAY(), DAY)

2. Date Category

Categorizes chassis based on days since arrival

Original Formula (SQL)
case
    when [Date Diff] between 0 and 60 then 'Less than 60 days'
    when [Date Diff] > 60 then 'More than 60 days'
end
DAX Formula
IF(
    [Date Diff] >= 0 && [Date Diff] <= 60,
    "Less than 60 days",
    IF([Date Diff] > 60, "More than 60 days")
)
